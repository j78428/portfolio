福泰旅遊 網站改版

UI UX Web Design

產品名稱：福泰旅遊公司網站

擔任角色：UI / UX Designer

負責內容：資訊架構整合、網站設計


1. 概要說明

過去因為不同的產品類型及策略，所以除了公司官網以外，也陸續建立了許多不同業務的網站，瀏覽體驗也越來越不好，所以這次改版目的除了翻新已不合時宜的網頁設計，還要整併所有產品集中在同一個網站進行銷售。

舊網站設計 - 首頁

舊網站設計 - 訂票網站


2. 設計目標

  • 統合不同類型的產品搜尋功能，讓搜尋功能可以更方便。
  • 承襲品牌視覺元素，定義色彩計畫，翻新網站設計。

3. 設計結果

改版網站

細部說明
  • 調整色彩應用邏輯，將醒目的的品牌橘色，用於局部需要凸顯標記的項目上，例如：標示、按鈕。
  • 搜尋項目採用隔線區分項目，減少過多複雜層級的堆疊。
  • 整併不同產品類型的搜尋條件。
  • 取消裝飾性的設計元素，更多以單純的圖片素材作呈現。
